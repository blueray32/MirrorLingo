# Exercise: Build Commands with Output Patterns

**Duration**: 30-40 minutes
**Purpose**: Apply command patterns by building a context loading command and a document creation command optimized for their consumers

---

## Overview

You'll build two commands demonstrating the core patterns from Video 2:

1. **Exercise 1: Priming Command** (Pattern 1: Context Loading)
   - Load tool development patterns into agent context
   - Output optimized for YOU (human-readable summary)

2. **Exercise 2: Planning Command Output** (Pattern 2: Document Creation)
   - Design structured output for feature planning
   - Output optimized for ANOTHER AGENT (structured, explicit)
