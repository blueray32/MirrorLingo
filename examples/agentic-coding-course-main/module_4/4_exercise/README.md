## Overview

You'll build two commands demonstrating the core patterns from the last video:

1. **Exercise 1: Priming Command** (Pattern 1: Context Loading)
   - Load tool development patterns into agent context
   - Output optimized for YOU (human-readable summary so you can validate understanding)

2. **Exercise 2: Planning Command Output** (Pattern 2: Document Creation)
   - Design structured output for feature planning
   - Output optimized for ANOTHER AGENT (structured, explicit)
